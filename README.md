# ABENICS: Active Ball Joint Mechanism with 3DoF

# **Project info**

This project is to implement ABENICS mechanism which has 3 degree of freedom only in one joint. 

## **The components are**

- Two Driving Module
    - Monopole Gear
    - Pinion
    - Differential Inner Worm Gear
    - Helical Gear
    - Coupling
    - Gear Holder
- One Spherical Joint
    - Output Link Integration
    - Gear Holder
